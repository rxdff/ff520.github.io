写给自己的Markdown使用指南

昨天搞完了Hexo，并选了一个Theme，今天来学习下Markdown使用。

Markdown介绍

Markdown 是一种用来写作的轻量级「标记语言」，它用简洁的语法代替排版，而不像一般我们用的字处理软件 Word 或 Pages 有大量的排版、字体设置。它使我们专心于码字，用「标记」语法，来代替常见的排版格式。例如此文从内容到格式，甚至插图，键盘就可以通通搞定了。目前来看，支持 Markdown 语法的编辑器有很多，包括很多网站（例如简书）也支持了 Markdown 的文字录入。Markdown 从写作到完成，导出格式随心所欲，你可以导出 HTML 格式的文件用来网站发布，也可以十分方便的导出 PDF 格式，这种格式写出的简历更能得到 HR 的好感。甚至可以利用 CloudApp 这种云服务工具直接上传至网页用来分享你的文章，全球最大的轻博客平台 Tumblr，也支持 Mou 这类 Markdown 工具的直接上传。

3.列表

3.1 有序列表

数字不能省略但可无序，点号之后的空格不能少。建立多级列表时，每个子级别都要比上个级别前面多打2个以上空格。

  1. Item-1
    1.1. Item-1-1
    1.1.1. Item-1-1-1
    1.1.1.1. Item-1-1-1-1
    1.2. Item-1-2
  2. Item-2
  3. Item-3
    1. Item-3

3.2 无序列表

符号之后的空格不能少，-、+、*效果一样，在嵌套列表中可以循环使用。

  • Item-1
    • Item-1-1
      • Item-1-1-1
        • Item-1-1-1-1
    • Item-1-2
  • Item-2
    • Item-2-1
      • Item-2-2
        • Item-2-3
  • Item-3

4.链接

以中括号[]标记显示的链接文本，后面紧跟用小括号()包围的链接。

百度

也可以使用<>包裹内容，但要注意内容一定要符合URL或者邮箱规则才行，否则不会显示出来。

https://www.baidu.com

<www.baidu.com>

4.图片

添加图片的使用方法基本上和链接类似，只是在中括号前加叹号!，即![图片标题](图片地址)。

6.表格

个人感觉表格书写是最麻烦的

Tables Are Cool

zebra stripes are neat $1
  • |、-、:之间的多余空格会被忽略，不影响布局。
  • 默认标题栏居中对齐，内容居左对齐。
  • -:表示内容和标题栏居右对齐，:-表示内容和标题栏居左对齐，:-:表示内容和标题栏居中对齐。
  • 内容和|之间的多余空格会被忽略，每行第一个|和最后一个|可以省略，-的数量至少有一个。

10.hexo上传MD文件遇到的问题

  • YAMLException: end of the stream or a document separator is expected at line x, column y:

    出现这种情况，一般都是缺少空格，在:冒号之后要有空格！检查x行y列附近的冒号，其之后是否跟了空格。

    如果找不到问题，可尝试重装hexo的theme操作，白白浪费了2个小时，果然重启大法才是王道。
